
## Installation

To get started add the following link tag to your head element of the HTML file.

```bash
  <link rel="stylesheet" href="https://main--eager-nightingale-ec3f88.netlify.app/components/component.css">
```
## Components provided
- Cards
- Buttons
- Avatar
- Badges
- Alerts
- Images
- Input
- List
- Rating
- Toast
- Modal
